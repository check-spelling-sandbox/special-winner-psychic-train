Appendix C. HA cluster

This appendix describes the Neo4j HA feature.

Please note that Causal Clustering is the preferred method for clustering in Neo4j. This is where investments in feature improvements and development are being placed.

While the HA feature is still fully supported for existing deployments, new installations should only use it together with Neo4j in embedded mode, or if specifically advised by Neo4j Professional Services.
